<h1>AntiDote - An Antivirus</h1>

 ### [YouTube Demonstration](https://www.youtube.com/channel/UC-9yN-mbnbmrlXxQUeRcDUw)

<h2>Description</h2>
Project consists of a simple Python script that uses the concept of hashing and provides features like scanning, Removing files containing viruses.
It also has a utility of Increasing RAM health and a service of Junk file Remover.
<br />


<h2>Languages and Utilities Used</h2>

- <b>Python</b> 
- <b>Virustotal for hashes</b>

<h2>Environments Used </h2>

- <b>Kali Linux</b>
